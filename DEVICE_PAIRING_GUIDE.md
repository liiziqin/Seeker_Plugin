# Android 無線偵錯配對指南 (Wi-Fi ADB Guide)

本指南說明如何透過 Android 11+ 的「無線偵錯」功能，在不連接 USB 線的情況下與電腦進行配對並建立 Appium 控制連線。

## 📱 手機端設定步驟

1.  **進入開發者模式**：
    *   開啟「設定」 -> 「關於手機」 -> 連續點擊「版本號碼」直到顯示已啟動開發人員選項。
2.  **開啟無線偵錯**：
    *   進入「設定」 -> 「系統」 -> 「開發人員選項」。
    *   找到「**無線偵錯**」選項。
    *   **重要：** 不要只打開開關，請直接 **點擊「無線偵錯」文字部分** 進入詳細設定頁面。
3.  **啟動配對模式**：
    *   點擊「**使用配對碼配對裝置**」。
    *   畫面會顯示：
        *   **IP 地址和埠號** (例如：`192.168.1.10:33455`)
        *   **Wi-Fi 配對碼** (6 位數字)

---

## 💻 電腦端操作步驟

1.  **執行配對指令**：
    *   開啟命令提示字元 (CMD) 或 PowerShell，輸入手機畫面上顯示的 IP 與埠號：
    ```bash
    adb pair [IP]:[Port]
    # 範例：adb pair 192.168.1.10:33455
    ```
2.  **輸入配對碼**：
    *   當命令列出現 `Enter pairing code:` 時，輸入手機畫面上顯示的 **6 位配對碼**。
3.  **確認連線狀態**：
    *   配對成功後，通常會自動連線。請再次輸入以下指令確認：
    ```bash
    adb devices
    ```
    *   若看到裝置清單中出現 `[IP]:[Port] device`，代表連線成功！

---

## 🛠️ 常見問題處理
*   **指令無效**：請確保已安裝 Android Platform Tools，且 `adb` 已加入系統環境變數。
*   **無法配對**：請確認手機與電腦連線至 **同一個 Wi-Fi 網路**。
*   **連線埠號變化**：配對用的埠號 (Pairing Port) 與連線用的埠號 (Connect Port) 通常不同，配對成功後若中斷，請查看「無線偵錯」主頁面上顯示的「IP 地址和埠號」進行連線：
    ```bash
    adb connect [IP]:[Connect_Port]
    ```
