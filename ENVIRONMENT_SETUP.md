# 環境設定 (Environment Setup)

## Node.js 版本
請使用 Node.js `v24.13.1`。

## npm 版本
請使用 npm `11.10.0`。

### 升級指令
如果你使用的 npm 是舊版本，請執行以下指令進行升級：

```bash
npm install -g npm@latest
```

## Appium 安裝
請執行以下指令安裝 Appium：

```bash
npm install -g appium
```

### 驗證版本
安裝完成後，請確認版本為 `3.2.0`：

```bash
appium -v
```

## UIAutomator2 驅動安裝
請執行以下指令安裝 UIAutomator2：

```bash
appium driver install uiautomator2
```
